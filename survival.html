Survival Analysis

Survival Analysis considers the likelihood of an event with time. It is used to model the interval taken to reach the predicted outcome given an initial set of parameters. This type of analysis is extremely powerful and is used to great effect in medicine, for example survival analysis can be used to predict the lifespan of someone who smokes tobacco. Another example is comparing lifespans of patients given different treatments.

An extremely powerful type of survival analysis uses proportional hazards to determine survival likelihood when the model data is incomplete or "censored".
